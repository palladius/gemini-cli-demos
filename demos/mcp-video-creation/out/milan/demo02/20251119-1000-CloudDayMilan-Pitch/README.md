# Cloud Day Milan - Teaser Video

This folder contains the assets for a teaser video for the Cloud Day Milan event, created on 2025-11-19.

The video is inspired by a "Men in Black" theme, where developers are portrayed as secret agents fighting against technological "monsters".

## Storyboard

The storyboard for the video can be found in the `STORYBOARD.md` file.

## Assets

The assets for each scene are located in this folder and are named according to the scene number.
